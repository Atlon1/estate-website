#First e-state shop.
The first store, A page where you will find the opportunity to buy an apartment / house. With an advanced search module by specified criteria.

#Features
- Responsive design
- Props validations
- React Scroll
- Animations
- Advanced filter bar


#Technologies
- Tailwind css
- RWD
- Typescript
- Fetch: GET, POST
- React: react-router
- Context API
- UseParams
- Used big data-base.
